#--
Year: 1969
Title: The C programming language.
Text: Dennis Ritchie and Ken Thompson develops the C programming language (not the book).
C is an imperative (procedural) structured, ALGOL fashioned language. Although C might have
been developed mainly for the UNIX operation system, it has gotten extremely popular and
compilers have been made for almost all available platforms since.
#--
Year: 1969
Title: UNIX
Text: Ken Thompson and Dennis Ritchie (including others) started developing a new operating
system at AT&Ts Bell Labs. The first version of the UNIX operating was developed for the
PDP-11 system replacing CP/M.
#--
Year: 1978
Title: The beginning of the C++ programming language.
Text: The Danish computer scientist Bjarne Stroustrup began developing C++ (then called
"C with classes"). One of the main differences between C and C++ is C++ object oriented
nature, and its rich standard library.
#--
Year: 1980
Title: Usenet gets established
Text: Tom Truscott and Jim Ellis establishes Usenet. Usenet lets users read and post
messages categories, known as newsgroups. The main difference between Usenet and a traditional
web forum is the absence of a central server structure and no administrator role. Access
to Usenet is operated by a commercial usenet provider, an ISP, university or other means of
access.
#--
Year: 1987
Title: Perl programming language.
Text: Larry Wall develops the first version of the Perl programming language. Perl was
originally developed as a general-purpose Unix scripting language.
#--
Year: 1991
Title: GPL version 2
Text: GNU GPL (General Public License) version 2 releases.
#--
Year: 1991
Title: The Linux kernel
Text: Linus Torvalds releases the initial version of the Linux kernel, first named the
freax, but renamed Linux by Ari Lemmke. Torvalds had initially considered the name Linux,
but dismissed it too egotistical.
#--
Year: 1991
Title: PGP
Text: Phil Zimmerman releases the PGP (Pretty Good Privacy) encryption package. PGP was
an easy way for the masses to encrypt communication using RSA or DSA algorithms.

Text: Tim Berners-Lee founds W3C (The World Wide Web Consortium) after he leaves CERN in
October 1994.
